## Compilación y ejecución
Para compilar el programa se debe ejecutar el siguiente comando:
```
g++ main.cpp && ./a.out  
```
### Ejecutar daemon
Para ejecutar el daemon en la mini shell se debe ejecutar el siguiente comando:
```
daemon <t> <p> 
```
